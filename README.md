# 🌟 RIFLUX Modpack - Infinite Ascension

**A Minecraft Forge 1.20.1 Modpack inspired by the RIFLUX game concept**

## 🎮 Overview

RIFLUX transforms Minecraft into an epic RPG experience with:
- **Infinite Progression**: Level up indefinitely with no level cap
- **Dynamic Economy**: Trade with MBAQ currency system
- **Multiple Worlds**: Explore diverse biomes and dimensions
- **Magic & Combat**: Advanced spell systems and epic battles
- **Character Development**: Choose from 6 unique classes

## 📦 Quick Start

1. Clone this repo
2. Copy to your Minecraft mods folder
3. Install Forge 1.20.1
4. Launch and enjoy!

## 🎯 Features

### Level System
- Levels 1-20: Fast progression
- Levels 20-60: Moderate progression  
- Levels 60+: Slow progression
- Rebirth System: Reset for power multiplier

### MBAQ Economy
- Starting Balance: 100 MBAQ
- Earn from mobs, quests, trading
- Use for crafting and enchanting

### Character Classes
- Warrior | Mage | Archer | Rogue | Paladin | Druid
